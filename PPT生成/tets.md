- 股权经营
```
股权经营：取当前周上一双周推进会的项目数据。
```

- 董监高管理
```
经营目标达成率：取外派高管的所有参股公司经营情况里的利润总和，目标值由自己设定算出达成率。

履职报告提报率：取所有外派高月报履职评价（董事）结束状态占总条数的比。

专项管理建议提报：取外派高月报履职评价里（董事）项管理建议个数。

委派人数：取所有委派的一汽方且任期内(提名方为股权或资产公司)高管、董事、监事人数。

人员配置率：取所有委派的一汽方且任期内(提名方为股权或资产公司)高管、董事、监事人数占所有高管、董事、监事占比。

参股公司覆盖率：取存在一汽方委派且任期内的参股公司与所有参股公司的比。

履职参会次数：取所有已结束的三会流程中议题提报-议题资料传达勾选过的董事、监事参会次数。
```

- 三会管理
```
三会柱状图：取三会计划里录入的所有议题按月度、和审批层级放到相应的柱子里。点击即可跳转到相应详情页、议题名称可以点击弹窗展示议题模型名称、审批层级、总办会（分管领导、总监优先级）决议信息、三会决议信息。

计划完成情况：展示该年度所有三会计划的议题，最后一列给状态。

是否完成：取提报议题关联了计划议题且走完议题审核步骤。

审议议题数量：三会年度计划所有议题走完议题审核步骤。

计划逾期议题：新增议题时日期晚于关联议题的三会计划里的会议召开日期的所有议题。

否决议题占比：三会计划里议题在表决建议被任一会议不同意的议题占所有议题占比。
```

- 参股党建
```
党组织覆盖率：建立党组织的参股公司数量/总参股公司数量。
党建进章程率：已将党建工作要求写入公司章程的参股公司数量/总参股公司数量。
```
